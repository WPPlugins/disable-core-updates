== Description ==

A simple WordPress plugin that disables all WordPress core updates.

Automaticly disables the updates and removes the WordPress core updating interface.


= Remove Updates Submenu =
If you would like to remove the updates submenu in your dashboard. 
Download our new plugin: <a href="http://wordpress.org/plugins/remove-the-updates-submenu/">Remove the Updates Submenu</a>.

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
